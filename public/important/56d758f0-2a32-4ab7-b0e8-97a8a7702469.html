  1. HTML uses text-based tags for placing images, tables, links, and other webpage elements.
  2. It provides the essential structure and content for any webpage.
  3. Web browsers interpret HTML code to display webpages to users.
  4. Hypertext means "text within text," enabling links between webpages.
  5. HTML became an Internet standard in 1993.
  6. It is supported by all major browsers, ensuring uniform webpage appearance.
  7. HTML uses tags (e.g., <p>, <img>) to define elements and their attributes.
  8. It defines both the visual layout and semantic meaning of webpage elements.
  9. Designed for machine interpretation, HTML is also simple enough for humans to read.
  10. Forms the foundation of all webpages found on the Internet.
  11. Used extensively in education, email, banking, and web-based applications.
  12. A fundamental technology that every modern professional should understand.
Page 5
  1. HTML uses tags to instruct browsers on how to display text.
  2. Tags like `<b>` create bold text; `<i>` makes text italicized.
  3. HTML pages are made with text editors and saved with a .html extension.
  4. Basic HTML structure includes DOCTYPE Declaration and an HTML Section.
  5. The HTML Section further encompasses the HEAD and BODY components.
  6. DOCTYPE Declaration is the first line, informing the browser of HTML version.
  7. All HTML code is enclosed within `<html>` and `</html>` tags.
  8. The HTML tag marks the start and end of the entire document.
  9. Every HTML element is nested within the main `<html>` tags.
  10. Closing tags always use a slash ( / ), for example, `</html>`.
  11. The `<head>` tag is enclosed between `<head>` and `</head>` tags.
  12. The head tag contains metadata, information not directly visible on the page.

  1. 3D Graphics focuses on creating and managing three-dimensional visual representations, a key role in computer graphics.
  2. Digital cameras spurred rapid growth in software for image editing and manipulation.
  3. Image manipulation tools allow removing blemishes or 'red-eye' from photos.
  4. These tools enable cutting and pasting specific parts of images.
  5. Users can create composite or unreal images with manipulation software.
  6. Computer graphics are used in film and TV for special effects, like removing wires or overlaying images.
  7. The transition from analog film to digital imaging was driven by these applications.
  8. 2D graphics applications include tools like Microsoft Paint for simple drawings.
  9. Basic 2D features involve drawing dots, lines, and shapes such as rectangles and ovals.
  10. 2D graphics tools also allow filling regions with various colors.
  11. Cutting and pasting image sections is a common feature in 2D applications.
Page 17
  1. 2D Graphics produces images by converting shapes into pixel patterns.
  2. Image Processing analyzes existing images to detect or enhance patterns.
  3. Image Processing is commonly used in Artificial Intelligence (AI) applications.
  4. 2D Graphics focuses on creating images; Image Processing focuses on understanding them.
  5. 3D Graphics creates 3D virtual scenes, converting them into 2D images to simulate photography.
  6. Unlike real photography, 3D graphics images virtual worlds existing only as data.
  7. Scene Creation & Encoding is the artistic step of designing the 3D environment.
  8. Image Production is the computational step of generating the final rendered image.
  9. 3D graphics are widely used in interactive video games for virtual world interaction.
  10. Animated movies use 3D graphics, allowing animators to create frames in virtual environments.
Page 18
  1. Current media like movies and games still use 2D displays for 3D worlds.
  2. Future users may interact directly with 3D virtual worlds, not just view them.
  3. Potential for 3D graphics projectors and immersive storytelling for viewers.
  4. Immersive storytelling could allow viewers to explore or change scenarios.
  5. Advances in 3D human–machine interfaces will enable interactive virtual experiences.
  6. The 3D graphics process involves three main steps: Modeling, Rendering, and Displaying.
  7. Modeling designs and constructs a 3D scene using digital data and algorithms.
  8. Modeling is analogous to building a physical set in the movie industry.
  9. The created 3D scene during modeling may not exist in the real world.
  10. Rendering converts the 3D model into a 2D image from a specific camera position.
  11. Rendering involves applying mathematics and analytic techniques to process models.
Page 19
  1. Geometry is used to project objects onto a projection plane.
  2. Perspective Projection projects objects from a common viewpoint along lines (projectors).
  3. Perspective Projection produces images that are similar to human vision.
  4. Parallel Projection uses parallel projectors, showing a true profile of objects.
  5. Parallel Projection is commonly utilized in engineering drawings for object representation.
  6. The image window defines the final image boundaries on the projection plane.
  7. Only content within the image window appears, like a camera's viewfinder frame.
  8. Rendering requires computing the appearance of each individual pixel in the image.
  9. Pixel computation is complex due to lighting, material properties, and object interactions.
  10. It accounts for lighting intensity/direction, material properties, and object interactions like shadows.
  11. Pixel computation relies on principles from physics and material science.
  12. Computed pixels are stored as a bitmap in a frame buffer.
Page 20
  1. The frame buffer is a special memory area used for storing image data.
  2. Frame buffers can be part of main memory or dedicated graphics memory (GPU memory).
  3. Final images in the frame buffer are displayed immediately for real-time rendering (games, simulators).
  4. Alternatively, frame buffer images can be stored for later use, such as in movies or animations.
  5. Real-time rendering limits image quality due to inherent processing constraints.
  6. Consequently, animated movies generally achieve higher visual quality than video games.
  7. A video game represents an encoded 3D virtual world manipulated by software.
  8. As a player interacts, the scene is repeatedly rendered and stored in the frame buffer.
  9. Special-purpose graphics hardware (GPU) is responsible for handling real-time rendering in games.
  10. The display shows updated images from the frame buffer, creating the illusion of movement.
  11. Game consoles differ from PCs primarily due to the presence of dedicated graphics hardware.
